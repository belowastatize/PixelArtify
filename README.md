# PixelArtify

![PixelArtify Banner](https://via.placeholder.com/900x200?text=PixelArtify+Banner)

## Overview
PixelArtify is an intuitive web application designed for both beginners and professionals to create, edit, and animate pixel art. Whether you're designing sprites for games or just having fun, PixelArtify offers a clean interface and powerful tools right in your browser.

## Features
- Draw pixel-perfect art on a customizable canvas.
- Create animations frame-by-frame with onion skinning.
- Export your artwork as PNG or animated GIF.
- Multiple brush sizes and color palettes.
- Undo/redo functionality.
- Responsive and accessible design.

## Tech Stack
- **Frontend:** React, TypeScript, Tailwind CSS
- **Animation Engine:** Custom canvas manipulation with HTML5 Canvas API
- **Deployment:** Vercel

## Installation
Clone the repository:

```bash
git clone https://github.com/belowastatize/PixelArtify.git
cd PixelArtify
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Usage
1. Launch the app and select your canvas size.
2. Use the tool palette to draw pixel art.
3. Add animation frames as needed.
4. Preview your animation with the play controls.
5. Export your artwork using the "Export" button.
